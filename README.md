# ⛅ Weather App

A simple Weather App built using Python and Streamlit that provides real-time weather information using the OpenWeatherMap API.

## Features

- 🌡️ Temperature
- 💧 Humidity
- 🍃 Wind Speed
- ☁️ Weather Condition
- 📅 Current Date
- 🔍 Search weather by city

## Technologies Used

- Python
- Streamlit
- OpenWeatherMap API
- Requests
- Python-dotenv

## Installation

1. Clone this repository.
2. Install the required packages:

```bash
pip install -r requirements.txt
```

3. Create a `.env` file and add your OpenWeatherMap API key:

```
WEATHER_API_KEY=your_api_key
```

4. Run the app:

```bash
streamlit run app.py
```
